# Omie Laravel

Pacote de integração com [https://omie.com.br/](https://omie.com.br/)

## Instalando

Instale com [composer](https://getcomposer.org/):

```bash
composer require bee-delivery/omie
```

## Como utilizar?

[Documentação API Omie](https://developer.omie.com.br/)
